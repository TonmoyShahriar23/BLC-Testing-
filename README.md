BLC: E-Learning Web Portal Testing Project

BLC is a comprehensive manual web testing project conducted on an educational Learning Management System (LMS).
The purpose of this project is to evaluate the functional behavior, usability, and reliability of the web portal by identifying defects, validating core features, and ensuring a smooth user experience for students.

The testing focuses on critical academic functionalities used in day-to-day learning activities and aims to ensure that the system behaves correctly under both normal and abnormal user actions.

The following major modules were tested:

Login & Authentication

Dashboard

Course List

Course Content (Video, PDF, Lecture)

Assignment Submission

Scope of Testing

This project covers manual functional testing and includes:

Verification of core user functionalities

Validation and negative testing

UI and usability testing

Authorization and access control checks

Bug identification and documentation

The testing was performed from a student user perspective.
